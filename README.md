# Recipe Hub Client

## Project Overview

Recipe Hub Client is a modern recipe sharing web application where users can explore recipes, view recipe details, and enjoy a clean food-related browsing experience. The project focuses on responsive design, user-friendly navigation, and modern frontend development practices.

## Live Project Link

[Visit Live Website](https://recipe-hub-client-alpha.vercel.app/)

## Repository Link

[GitHub Repository](https://github.com/WiZzard14/recipe-hub-client)

## Technologies Used

- React
- TypeScript
- JavaScript
- Tailwind CSS
- Firebase
- Vercel

## Core Features

- Responsive recipe browsing interface
- Clean and modern UI design
- Recipe details display
- User-friendly navigation
- Fast loading performance
- Mobile, tablet, and desktop friendly layout

## Dependencies Used

- react
- react-router-dom
- typescript
- tailwindcss
- firebase
- react-icons
- sweetalert2

## How to Run the Project Locally

1. Clone the repository:

```bash
git clone https://github.com/WiZzard14/recipe-hub-client.git
```

2. Go to the project folder:

```bash
cd recipe-hub-client
```

3. Install dependencies:

```bash
npm install
```

4. Run the project:

```bash
npm run dev
```
